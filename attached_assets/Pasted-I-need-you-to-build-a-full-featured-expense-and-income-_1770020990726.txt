### Supabase Integration:
```javascript
// Initialize Supabase client
import { createClient } from '@supabase/supabase-js'

const supabase = createClient(
  process.env.SUPABASE_URL,
  process.env.SUPABASE_ANON_KEY
)

// Example: Fetch transactions
const { data, error } = await supabase
  .from('transactions')
  .select('*, categories(*), payment_methods(*)')
  .eq('user_id', user.id)
  .order('transaction_date', { ascending: false })

// Example: Add transaction
const { data, error } = await supabase
  .from('transactions')
  .insert({
    user_id: user.id,
    type: 'expense',
    amount: 50.00,
    category_id: categoryId,
    description: 'Lunch'
  })

// Example: Upload receipt
const { data, error } = await supabase.storage
  .from('transaction-attachments')

```

## Important Notes:

1. **Row Level Security**: All Supabase queries automatically filter by authenticated user due to RLS policies - you don't need to manually filter by user_id in SELECT queries
2. **Real-time Updates**: Consider using Supabase real-time subscriptions for live updates
3. **Error Handling**: Show user-friendly error messages
4. **Loading States**: Show skeletons or spinners while data loads
5. **Form Validation**: Validate all inputs before submission
6. **Responsive**: Must work perfectly on mobile devices
7. **Performance**: Optimize queries, use pagination for large lists
8. **Accessibility**: Follow WCAG guidelines
